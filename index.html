# Dictionary bookmarklet The __Dictionary__ bookmarklet allows you to double-click on words to translate from French. To [install the bookmarklet](https://raw.github.com/dergachev/pin-screenshot-bookmarklet/gh-pages/installation-screencast.gif), drag the following link to your bookmark toolbar:
Fr->En
## How to use it * Activate the bookmarklet * Double click on a french word ## Dev resources * http://glosbe.com/a-api * http://jsfiddle.net/timdown/krtTD/10/ * http://jsfiddle.net/kkgEt/2/ ## Get the source code To get the source code, visit https://gist.github.com/dergachev/e9339d65c1bf7a6c6226 For development purposes, the following bookmarklet injects `http://localhost:9090/dictionary-bookmarklet.js`: Fr-En:9090
